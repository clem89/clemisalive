### 레벨업 & 강화 선택 시스템
- **LevelSystem.cs** (신규): XP/레벨 관리 싱글턴. 적 처치 시 XP +10, 레벨업 필요량 50→70→90... 증가
- **LevelUpUI.cs** (신규): 레벨업 시 3개 강화 선택지 패널 표시. 버튼 클릭 시 강화 적용 후 게임 재개
- **Character.cs** (수정): 런타임 스탯 필드 추가 (`_rtMoveSpeed`, `_rtFireRate`, `_rtBulletDamage` 등), `ApplyUpgrade(UpgradeType)` 메서드로 8종 강화 적용
- **Bullet.cs** (수정): `Init(damage, speed, pierce)` 메서드 추가로 런타임 스탯 주입
- **BulletPool.cs** (수정): 스탯 전달 오버로드 `Get(pos, rot, damage, speed, pierce)` 추가
- **Enemy.cs** (수정): `Die()`에서 `AddKill()` → `OnEnemyKilled()` 로 변경 (XP 연동)
- **GameManager.cs** (수정): LevelUpUI 참조, `TriggerLevelUp()` / `OnUpgradeSelected()` / `PickRandomUpgrades()` 추가
- **HudUI.cs** (수정): XP 슬라이더, 레벨 텍스트 추가
