# 강화 선택지 기획

## 기본 규칙
- 매 레벨업마다 8종 중 **3개 랜덤** 제시
- 플레이어가 1개 선택 → 즉시 적용

## XP 설계
- 적 처치 1킬 = XP +10
- 레벨업 필요 XP: 50, 70, 90, 110, ... (매 레벨 +20)

## 강화 종류 (8종)

| ID | 이름 | 효과 | 비고 |
|----|------|------|------|
| MaxHp | 체력 증가 | MaxHP +20, 즉시 그만큼 회복 | 중첩 가능 |
| HpRestore | HP 회복 | 현재 MaxHP의 30% 즉시 회복 | 선택 즉시 소모 |
| MoveSpeed | 이동속도 | 이동속도 +10% | 중첩 가능 |
| FireRate | 공격속도 | 발사 딜레이 -15% | 중첩 가능 |
| BulletDamage | 총알 데미지 | 총알 데미지 +20% | 중첩 가능 |
| BulletSpeed | 총알 속도 | 총알 속도 +15% | 중첩 가능 |
| Pierce | 관통 +1 | 총알이 적 1마리 추가 관통 | 중첩 가능 |
| DashCooldown | 대시 쿨다운 | 대시 쿨다운 -20% | 중첩 가능 |

## 향후 추가 후보
- 멀티샷 (총알 동시 발사 수 증가)
- 총알 크기 증가
- 흡혈 (데미지의 일부 HP 회복)
- 폭발 (총알 소멸 시 범위 데미지)
- 이동속도 연동 데미지
