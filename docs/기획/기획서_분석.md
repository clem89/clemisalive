# 던전 러너 (Dungeon Runner) - 기획서 분석

> 원본 문서: `docs/기획/2주_로그라이크_기획서.docx`

## 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 탑다운 액션 로그라이크 |
| 플랫폼 | PC (키보드/마우스) |
| 제작 기간 | 2주 |
| 아트 스타일 | 심플한 2D 픽셀 또는 기하학적 도형 |
| 목적 | 개인 취미 프로젝트 |

## 게임 루프

```
던전 입장 → 랜덤 방 진입 → 전투/이벤트 → 보상 선택 → 다음 방 선택(3~4경로) → 보스 처치
                                                                              ↓ 사망 시
                                                                          처음부터 리스타트
```

## 구현 시스템 목록 (우선순위 순)

### 1. 핵심 전투 시스템 (4일)

**플레이어**
- WASD 이동 + 마우스 방향 조준 + 클릭 공격
- 기본 무기: 총알형 투사체 발사 (연사, 데미지, 관통 등 스탯화)
- 체력 + 대시: 간단한 생존 메커닉

**적 AI 3종**
- 돌진형
- 원거리형
- 소환형

**보스**
- 보스 1종, 패턴 2~3개 + 텔레그래프 표시

### 2. 맵 생성 시스템 (2일)

- 고정된 방 레이아웃 프리팹 5~7개
- 노드 기반 맵 (Slay the Spire 방식)
- 총 12~15 노드 구성
- 방 타입: 일반 전투 → 정예 전투 → 상점 → 보스
- Scriptable Object 기반 노드 시스템

### 3. 로그라이크 진행 요소 (3일)

**전투 보상**
- 무기 업그레이드: 3개 중 1개 선택
- 패시브 아이템: 10종, 중첩 가능

**메타 진행**
- 영구 화폐로 시작 스탯 강화 (체력+10%, 공격+5% 등)
- 새 무기/캐릭터 언락: 2~3개

**난이도 스케일링**
- 방 진행할수록 적 체력/공격력 스케일링

### 4. UI/메뉴 (2일)

- 메인 메뉴: 시작, 업그레이드, 설정
- 인게임 UI: 체력, 현재 무기 스탯, 미니맵
- 보상 선택 UI: 카드 형식 (카드 게임 개발 경험 활용)
- 사망 화면: 획득 재화, 다시 시작

### 5. 사운드/밸런스 (2일)

- 기본 효과음: 공격, 피격, 사망
- 배경음악: 1~2트랙 (freesound.org, incompetech 활용)
- 플레이테스트 & 밸런싱

### 6. 폴리싱 여유 (1일)

- 버그 수정 및 폴리싱

## 차별화 포인트 (선택적 추가)

카드 게임 개발 경험을 활용한 아이디어:
- 보상을 "카드 덱" 개념으로 → 획득한 패시브를 카드처럼 시각화
- 시너지 시스템 → 특정 아이템 조합 시 특수 효과 (예: 불 관련 3개 = 폭발 데미지)
- "카드 버리기" 메커닉 → 상점에서 원치 않는 패시브 제거 가능

## 주스(Juice) 이펙트 목록

- 적 피격 시: 하얗게 번쩍 + 약간 뒤로 밀림
- 적 사망 시: 화면 쉐이크 + 파티클 + "쾅!" 효과음
- 레벨업 시: 캐릭터 주변 링 이펙트 + 소리
